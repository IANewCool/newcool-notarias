# NewCool Notarias - Tramites Notariales Chile

Portal educativo sobre notarias y tramites notariales en Chile.

## Demo

[https://notarias.newcool.io](https://notarias.newcool.io)

## Caracteristicas

- Buscador de notarias
- Calculadora de aranceles notariales
- Guia de tramites y documentos
- Escrituras y poderes
- Legalizacion de documentos

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
